# 模型监控工具使用说明

## 数据格式要求

准备CSV文件，包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| score | 模型分数 | 300-1000之间的数值 |
| label | 标签 | 0或1 |
| verify_time | 时间戳 | 10位时间戳，如1769932754 |
| fea1, fea2, ... | 特征列 | 数值型特征，列名可自定义 |

**示例数据：**
```
score,fea1,fea2,fea3,verify_time,label
665,50.0,1.4397,1.8830,1769932754,0
670,49.0,12.1532,8.3511,1769932843,0
680,50.0,7.9687,15.8507,1769932876,0
```

## 快速开始

### 1. 准备数据文件

将CSV文件放入`data`文件夹中

### 2. 运行监控

```bash
python main.py
```

### 3. 查看结果

监控报告会自动生成在`reports`文件夹中

## 参数说明

### ModelMonitor参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| score_col | 模型分数列名 | 必填 |
| label_col | 标签列名 | 必填 |
| feature_cols | 入模特征列名列表 | 必填 |
| timestamp_col | 时间戳列名 | 'verify_time' |
| week_col | 周数列名 | 'week' |
| n_bins | 分箱数量 | 10 |
| score_bins | 模型分数自定义分箱边界 | None（自动分箱） |
| feature_bins | 特征自定义分箱边界字典 | None（自动分箱） |
| binning_strategy | 分箱策略 | 'auto' |

### run方法参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| data_path | 数据文件路径 | 必填 |
| base_week | 基准周 | None（第一周） |
| output_excel | Excel输出路径 | None（不导出） |

### binning_strategy选项

- **'auto'**: 自动选择（先尝试等频分箱，失败则降级为等距分箱）
- **'equal_freq'**: 等频分箱（每个分箱样本数相近）
- **'equal_width'**: 等距分箱（每个分箱宽度相同）

## 输出说明

Excel报告包含以下sheet页：

1. **模型KS** - 模型区分度指标
   - week: 周数
   - ks: KS值
   - bad_cnt: 坏样本数
   - bad_rate: 坏样本率

2. **模型PSI** - 模型稳定性指标
   - week: 周数
   - psi: PSI值

3. **模型排序性** - 模型分数分箱统计
   - bin: 分箱编号
   - bin_range: 分箱范围
   - count: 样本数
   - percentage: 占比（每周总和为1）
   - group: 周数

4. **特征PSI** - 特征稳定性指标
   - feature: 特征名
   - week: 周数
   - psi: PSI值

5. **feaX排序性** - 每个特征的分箱统计
   - bin: 分箱编号
   - bin_range: 分箱范围
   - count: 样本数
   - percentage: 占比（每周总和为1）
   - group: 周数

6. **模型分与特征分箱策略** - 分箱边界和策略信息

## 智能分箱特性

### 自动降级机制

当数据分布不均匀时，程序会自动处理：

1. **等频分箱失败**：当唯一值数量小于分箱数时，自动降级为等距分箱
2. **分箱边界扩展**：自动将第一个边界设为-inf，最后一个边界设为inf，确保所有数据都能被分箱
3. **数据分布检测**：自动检测每个特征的数据分布情况，并在控制台输出警告

### 分箱策略选择建议

| 数据特点 | 推荐策略 | 说明 |
|---------|---------|------|
| 数据均匀分布 | 'equal_freq' | 每个分箱样本数相近，适合PSI计算 |
| 数据分布不均匀 | 'equal_width' | 每个分箱宽度相同，避免分箱失败 |
| 不确定数据分布 | 'auto' | 自动选择，智能降级 |
| 有业务规则 | 自定义边界 | 使用score_bins或feature_bins |



## 常见问题

**Q: 如何识别特征列？**
A: 需要手动指定feature_cols参数，程序不会自动识别。

**Q: 数据分布不均匀怎么办？**
A: 使用'auto'策略，程序会自动检测并降级为等距分箱。

**Q: 如何自定义分箱边界？**
A: 设置score_bins（模型分数）或feature_bins（特征）参数。

**Q: 周数如何计算？**
A: 以数据中最早的时间戳所在周为第1周，按7天为一周计算，格式为"YY-MM-DD ~ YY-MM-DD"。

**Q: percentage为什么每周总和为1？**
A: percentage是基于每周的总样本数计算的，反映该周内各分箱的占比。

**Q: 分箱边界显示-inf和inf是什么意思？**
A: 表示第一个分箱包含所有小于最小边界的数据，最后一个分箱包含所有大于最大边界的数据，确保所有数据都能被分箱。
